# RideSync

> Verified professional ride sharing — Lahore, Pakistan

---

## What is RideSync?

RideSync connects verified professionals and students who are already making a trip with others going the same way. Cost is split equally. Identity is verified. No strangers.

## Ride Types

- **CommuteShare** — Daily office rides, cost split, direct payment
- **CampusRide** — University rides, student-only via .edu.pk email
- **DriveDesk** — Discussion rides, book a professional's commute for a conversation
